# AI Cut

AI 图片背景移除工具。

## Status

🟢 Online

## Goal

通过 AI 模型自动移除图片背景，
帮助用户快速生成透明 PNG 图片。

## Features

- 图片上传
- AI 抠图
- 高清导出
- 图片编辑（旋转、放大缩小、背景颜色填充、裁剪）
- 证件照模式抠图（尺寸、颜色）
- 积分系统
- 历史抠图记录
